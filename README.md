# Chirpy Hugo Starter [![GitHub license](https://img.shields.io/github/license/geekifan/hugo-theme-chirpy.svg?color=blue)](https://github.com/geekifan/hugo-theme-chirpy/blob/main/LICENSE)

This repository uses the [Chirpy Hugo theme][chirpy] as a pinned Hugo module.

## Usage

Run:

```sh
hugo server --bind 0.0.0.0
```

## Dev Container

This repository includes a development container in `.devcontainer/` with Hugo and opencode installed.

Open it in VS Code/Codium and run the Hugo server on port `1313`.

## Theme

The theme is pinned in `go.mod` and resolved through Hugo modules.
